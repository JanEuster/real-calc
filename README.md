# Real Calculator(s)

## Instructions
### Prerequisites
- Git
- NodeJS

### 1. Clone Repository
```bash
foo@bar:~$ git clone https://github.com/janeuster/real-calc.git
```

### 2. Install Dependencies
```bash
foo@bar:~$ cd real-calc
foo@bar:~$ npm ci
```

### Run Dev Environment
```bash
foo@bar:~$ npm run dev
```

### Build React App
<sub>This is not the Electron Desktop App</sub>
```bash
foo@bar:~$ npm run build
```
